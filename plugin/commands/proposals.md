List pending proposals:

```bash
stratus learning proposals $ARGUMENTS
```

Shows proposals generated by the adaptive learning engine from detected patterns in the codebase. Each proposal includes a confidence score, type (RULE, ADR, TEMPLATE, SKILL, PROJECT_GRAPH), and summary.

Options:
- `--max-count <n>` — maximum proposals to show (default: 10)
- `--min-confidence <f>` — minimum confidence threshold (default: 0.0)
